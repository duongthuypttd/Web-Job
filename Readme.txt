Để chạy trang web cần sử dụng phần mểm XAMPP, mở XAMPP Control Panel, bấm chọn Start ở các module Apache, MySQL.

Vào đường dẫn "http://localhost:8080/phpmyadmin/" để đăng nhập vào phpmyadmin sử dụng user là "root", để trống mật khẩu, sau đó vào mục Import, chọn file "database.sql", sau đó nhấn Import để import database.

Để chạy được trang web cần lưu source code vào thư mục htdocs của phần mềm XAMPP, sau đó truy cập đường dẫn "http://localhost:8080/" để đi đến trang web.
